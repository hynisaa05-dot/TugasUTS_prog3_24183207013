Proyek ini dirancang sebagai aplikasi point of sale (POS) sederhana atau formulir input pesanan. Tujuannya adalah untuk mendata pesanan pelanggan berdasarkan nomor meja, nama pelanggan, kategori makanan, dan daftar menu yang dipilih. Analisis Komponen GUI (Swing Controls)

JFrame,JLabel,JTextField,JComboBox,JList / JTextArea,JButton(TAMBAH,HAPUS,PESAN)

Alur Kerja (Logika)
Pengguna mengisi identitas $\rightarrow$ memilih kategori & menu $\rightarrow$ klik Tambah $\rightarrow$ klik Pesan untuk finalisasi. Secara teknis, ini melibatkan event handling (ActionListener) untuk memproses input teks menjadi data sistem.


Proyek UTS.java ini adalah aplikasi GUI (Graphical User Interface) berbasis Java Swing yang berfungsi sebagai sistem input pemesanan makanan.
Secara teknis, proyek ini mengintegrasikan komponen input data (teks dan pilihan) dengan logika tombol aksi (Tambah, Hapus, Pesan) untuk mengelola data pesanan pelanggan. Fokus utamanya adalah mempermudah interaksi pengguna dalam mendata nomor meja, nama, dan menu yang dipilih secara terorganisir.
